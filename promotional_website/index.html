Smart Scheduling for Schools



Currently, Utah charter schools and secondary schools lack the technological resources for building adequate school schedules. Schools need a schedule that complies to student, school, and teacher needs, course requirements, and time blocks.


Schoolbloc is an open source web application framework for generating class schedules under all these constraints. We use a schedule constraint solver to create an optimized schedule under constraints such as classroom sizes, courses with assigned teachers, and student preferences.
